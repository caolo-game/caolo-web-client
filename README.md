## Prerequisites

-   [NodeJS V10](https://nodejs.org/en/)
-   [Yarn](https://yarnpkg.com/lang/en/)

## Building and running

-   Init
    ```
    yarn install
    ```
-   Run
    ```
    yarn start
    ```

Prettier issue:
Remove 'prettier.eslintIntegration' from prettier config or prettier can crash on react hooks.
